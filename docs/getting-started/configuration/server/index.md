# Server side configuration

- [Schema Store](schema-store) [Required] - Configure the schema store includes the schema definitions.
- [Plugin](plugin) [Optional] - Configure the plugin which can trigger the event before and after the operation.
- [Data Filter](data-filter) [Optional] - Configure the data filter which can filter the data based on custom rules.
- [Localization](localization) [Optional] - Configure the localization for time zone and language.
